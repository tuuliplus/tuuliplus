# tuuli.plus

https://tuuli.plus

## Getting Started

First, run the development server:

```bash
yarn dev
```

Then, open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

Run:

```bash
yarn build
```
